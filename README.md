# Resume Scanner & Job Matcher

## Description
This project scans a resume and a job description and calculates a matching score using Natural Language Processing (NLP).

## Technologies Used
- Python
- Flask
- NLP
- TF-IDF Vectorizer
- Cosine Similarity
- HTML, CSS

## How It Works
1. User uploads a resume
2. User provides a job description
3. The system extracts text from both
4. TF-IDF converts text into vectors
5. Cosine similarity calculates matching score

## Output
The application displays a percentage match showing how suitable the resume is for the given job role.

## Use Case
Helps recruiters quickly shortlist candidates based on skill matching.
